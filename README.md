# Student-report-card-using-C-
Student Report Card SystemThis is a simple console-based application developed in C as part of my BCA coursework. 

The program is designed to manage student records, calculate grades, and display report cards.

🛠 FeaturesAdd Records:

1. Enter student details including name, roll number, and marks for various subjects.

2. Calculate Grades: Automatically calculates total marks and percentages.

3. View Reports: Display a formatted report card for individual students 

🚀 How to RunDownload or clone the repository.Open the .cpp file in Dev-C++ or any C++ IDE.Compile and run the code ($F11$ in Dev-C++).Follow the on-screen menu instructions.📂 Technical DetailsLanguage: C++Concepts Used: Classes and Objects, File Handling (fstream), and Basic Logic.
